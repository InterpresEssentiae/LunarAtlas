<div align="center">
  <h2>ИСПОЛЬЗОВАНИЕ</h1>
</div>
<ol>
  <li>Скачайте и установите приложение Obsidian с <a href="https://obsidian.md">официального сайта</a>.</li>
  <li>Загрузите данный репозиторий, выбрав опцию `"Code" → "Download ZIP"`.</li>
  <li>В главном меню Obsidian выберите пункт `"Открыть папку как хранилище"` и укажите директорию `Essentia Mundi`.</li>
</ol>
